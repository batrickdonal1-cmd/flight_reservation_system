# ✈️ Flight Reservation System (Python + SQL + Kaggle Dataset)

## Overview
A simple college-level project that simulates a flight reservation system using Python, SQLite, and a Kaggle dataset.

## Features
- Search flights by source and destination
- Book tickets
- Store data in SQLite database
- Load real-world flight data from Kaggle

## Dataset
[Kaggle Flight Reservation Dataset](https://www.kaggle.com/datasets/ashishpandey2062/flight-reservation-dataset)
